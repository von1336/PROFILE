
**Python-разработчик**

## Обо мне
Разработчик с опытом создания веб-приложений полного цикла. Специализируюсь на построении отзывчивых пользовательских интерфейсов, проектировании серверной логики и интеграции внешних API. Нацелен на создание чистого, поддерживаемого кода и эффективных, масштабируемых решений.

## Проектный опыт

### Система распределения заявок с интеграцией Google Таблиц
- Разработал полнофункциональное веб-приложение для автоматизации workflow по обработке и распределению входящих заявок.
- Реализовал REST API на **Django REST Framework** для управления заявками, пользователями и статусами.
- Интегрировал **Google Sheets API** для двусторонней синхронизации данных: автоматический импорт новых строк из таблицы как заявок и экспорт обновлений статусов обратно в таблицу.
- Спроектировал и реализовал реляционную базу данных на **PostgreSQL** для хранения структурированных данных приложения.
- Создал интуитивно понятный пользовательский интерфейс с использованием **React** для отображения списка заявок, фильтрации, назначения исполнителей и отслеживания истории изменений.
- Настроил аутентификацию и авторизацию на основе ролей (JWT).

### Веб-приложение для управления задачами (Task Manager)
- Построил SPA-приложение с фронтендом на **Vue.js** и бэкендом на **Flask**.
- Реализовал CRUD-операции для задач и проектов через собственный REST API.
- Использовал **SQLite** в разработке с возможностью переключения на **PostgreSQL** для production.
- Добавил функции drag-and-drop для изменения статуса задач.
